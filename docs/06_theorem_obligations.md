# Theorem-Level Obligations

The conjecture program becomes mathematically meaningful only if the analytic
obligations are stated plainly.

## 1. Quaternionic Domain

Define the quaternionic domain, slice structure, regularity class, and branch
choices used by the lift.

## 2. Gamma-Lift Construction

Construct the quaternionic factorial/Gamma-lift operator with clear
compatibility to the complex Gamma function on selected slices.

## 3. Functional Equation Bridge

Relate the lift to the classical zeta functional equation, including Gamma
factors, powers of pi, conjugation, and the `s <-> 1-s` reflection.

## 4. Coherence Functional

Define `C(q)` without circular dependence on the desired critical-line
location. State its regularity, symmetry, and variational structure.

## 5. Stationarity-To-Zero Link

State and develop the exact implication connecting stationary states of `C(q)`
with nontrivial zeros of the Riemann zeta function.

## 6. Off-Slice Control

Define the coherence defect away from `Re(q)=1/2` and establish the sign,
coercive, or variational structure needed for the conjectural mechanism.

## 7. Diagnostic Agreement

Use zero data only as diagnostics for the model and as guidance for the
analytic development target.
